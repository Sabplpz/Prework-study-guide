HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The language is defined along with the html tag so that the browser knows the language
  • The verbiage often used to describe elements are Parent and Child Elements
  • The h1-6 tags are important for reasons such as people that use page readers

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • There is three different ways to use CSS, inline, internal style sheet and external style sheet
  • Declarations contain two components: the property and the value
  • You can choose colors by using ther HEX number, there's a website for it

GIT

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • The command line is where all git commands are written
  • cd ... will take you to the desired file or "menu", just make sure to follow the flow
  • code . will open VS Code

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • An array is a lits of numbered items, starting with 0
  • A function is a helpful way of saving lines of code, instead of repeating we just call it
  • A for loop is one of the loops that can be used in JavaScript to repeat while condition is met
